# 🚜 AgroChain v1.0 - Sistema Agroindustrial Automatizado

Bienvenido a **AgroChain**, una plataforma diseñada para la gestión inteligente de cultivos y comercialización agrícola mediante una economía propia basada en **Cordobad**.

## 🌟 Características Principales
* **Persistencia de Datos**: El nombre del usuario y su saldo de **Cacaos** se guardan en el navegador (`LocalStorage`), por lo que no se pierden al cerrar la página.
* **Economía Tokenizada**: Uso de la moneda interna "Cacaos" para el registro y adquisición de productos.
* **Automatización IoT**: Simulación de sensores de humedad para el monitoreo de cosechas en tiempo real.
* **Mercado Digital**: Registro de lotes de producción por parte de vendedores con actualización automática del inventario.

## 🛠️ Estructura del Proyecto
El sistema está construido con tecnologías web estándar para garantizar ligereza y velocidad:
* **HTML5**: Estructura semántica del dashboard.
* **CSS3**: Diseño responsivo con temática agroindustrial.
* **JavaScript (Vanilla)**: Lógica de negocio, manejo de sensores y persistencia de datos.

## 🚀 Instalación y Uso
1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. Configura tu perfil inicial con tu nombre y saldo de Cacaos.
